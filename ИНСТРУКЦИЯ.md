```bash

cat > /Users/ods/Downloads/AI/start.sh << 'EOF'
#!/bin/bash
cd /Users/ods/Downloads/AI
while true; do
  echo "$(date) Запуск Telegram бота..."
  node bot.js &
  TG_PID=$!
  echo "$(date) Запуск Мессенджер бота..."
  node messenger_bot.js &
  MSG_PID=$!
  wait $TG_PID
  kill $MSG_PID 2>/dev/null
  echo "$(date) Перезапуск через 5с..."
  sleep 5
done
EOF
chmod +x /Users/ods/Downloads/AI/start.sh

# Запустить
./start.sh
```

## 4. ОБНОВЛЕНИЕ OBSIDIAN МЕТАДАННЫХ

### Что такое метаданные в Obsidian

Каждая заметка начинается с frontmatter (YAML блок между `---`):

```yaml
---
type: email
subject: "Re: Статус проекта"
from: "ivan@auchan.ru"
date: 2026-05-26
folder: INBOX
source: imap
---
```

Бот использует эти поля для фильтрации и поиска.

### Как обновить метаданные вручную

```bash
cd /Users/ods/Downloads/AI

# Переиндексировать все заметки
node -e "
const {VaultIndex} = require('./indexer');
const idx = new VaultIndex('./ObsidianVault');
console.log('Заметок:', idx.stats().total);
console.log('Типы:', idx.stats().by_type);
"
```

### Как исправить заметки без метаданных

```bash
# Найти заметки без frontmatter
node -e "
const fs   = require('fs');
const path = require('path');
function findMd(dir, res=[]) {
  for (const e of fs.readdirSync(dir, {withFileTypes:true})) {
    const f = path.join(dir, e.name);
    if (e.isDirectory()) findMd(f, res);
    else if (e.name.endsWith('.md')) res.push(f);
  }
  return res;
}
const files = findMd('./ObsidianVault');
const noMeta = files.filter(f => !fs.readFileSync(f,'utf8').startsWith('---'));
console.log('Без метаданных:', noMeta.length);
for (const f of noMeta.slice(0,10)) console.log(' ', f);
"
```

## 5. КАК ПОЛУЧАТЬ ВСЕГДА СВЕЖУЮ ИНФОРМАЦИЮ

### Автоматически (уже настроено в боте)

| Источник | Частота | Действие |
|---|---|---|
| 📧 Почта | Каждый час | Новые письма → vault |
| 💬 Мессенджер | Каждый час | Новые сообщения → vault |
| 🎙️ Plaud | Каждый час | Новые транскрипции → vault |
| 💾 Яндекс Диск | Каждые 6 часов | Новые файлы → vault |

### Ручная синхронизация — Telegram бот

```
/sync          → меню выбора источника
/sync email    → только почта
/sync disk     → только диск
/sync all      → всё сразу
/refresh       → переиндексировать vault
```

### Ручная синхронизация — Яндекс Мессенджер

```
/sync email
/sync all
/refresh
```

### Ручная синхронизация — терминал

```bash
cd /Users/ods/Downloads/AI

# Только новые файлы
node syncer.js

# Принудительно все файлы (долго!)
node syncer.js --force

# Только почта
node -e "
const {syncEmail} = require('./syncer');
const cfg = require('./config');
syncEmail(cfg.VAULT_PATH).then(r => console.log('Новых:', r.new));
"

# Только диск
node -e "
const {syncDisk} = require('./syncer');
const cfg = require('./config');
syncDisk(cfg.VAULT_PATH).then(r => console.log('+'+r.new+' новых'));
"
```

### Vectra векторный индекс — когда пересоздавать

Если поиск работает плохо или ошибка `Malformed JSON`:

```bash
cd /Users/ods/Downloads/AI

# Удалить и пересоздать индекс
rm -rf .vectra_index
node bot.js
# Подождать 5-10 минут пока проиндексирует все 1800+ заметок
```
