# 🧠 Личная база знаний — что умеет система

## Из каких источников собираются данные

| Источник | Что забирается | Как |
|---|---|---|
| 📧 Яндекс Почта | Все письма, все папки | IMAP автоматически |
| 💬 Яндекс Мессенджер | Личные чаты + группы | Bot API polling раз в час |
| 💾 Яндекс Диск | .txt, .md, .json, .csv, .html файлы | Локальная синхронизация |
| 🎙️ Plaud Note | Транскрипции голосовых записей | Папка с .txt файлами |

---

## Что хранится в Obsidian Vault

Все данные превращаются в Markdown-заметки с wikilinks:

- **Каждое письмо** → отдельная заметка с метаданными (от кого, дата, тема)
- **Каждый чат** → один файл с хронологией сообщений, обновляется каждый час
- **Каждая запись Plaud** → заметка с транскрипцией, датой, количеством слов
- **Каждый контакт** → страница с историей всей переписки
- **000_MOC.md** → главная карта с навигацией по всему vault

## Что умеет Telegram-бот

Пишете боту в Telegram — он ищет по вашей базе и отвечает через Claude.

### Команды

| Команда | Что делает |
|---|---|
| Любой вопрос | Ищет по всей базе и отвечает (RAG) |
| `/summary` | Саммари почты / чатов / голосовых записей за последнее время |
| `/tasks` | Вытаскивает задачи и дедлайны из переписки |
| `/search текст` | Прямой поиск по заметкам |
| `/stats` | Статистика: сколько писем, чатов, записей |

### Примеры вопросов боту

- *«Что мы договорились с Иваном на прошлой неделе?»*
- *«Найди все упоминания проекта X»*
- *«Что я обсуждал с командой в марте?»*
- *«Есть ли дедлайны на этой неделе?»*
- *«Кратко: о чём была встреча 15 апреля?»* (если есть запись Plaud)

## Как обновляется база

| Источник | Частота |
|---|---|
| Яндекс Мессенджер | Каждый час автоматически |
| Почта | При каждом запуске `yandex_to_obsidian.py` |
| Plaud / Диск | Вручную (перезапустить скрипт) |

---

## Что НЕ умеет (текущие ограничения)

- ❌ Не читает вложения из писем (только текст)
- ❌ Не получает историю чатов до добавления бота
- ❌ Не работает с аудио напрямую — только уже готовые транскрипции
- ❌ Бот читает только чаты, в которые его добавили
